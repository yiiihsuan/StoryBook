# StoryBook
for AI-generative book

step
1. install node & npm first
2. git clone 
2. npm install
3. npm start to run the code


1. install node:

- install nvm first


-install node (node v18 for this project)
v18 LTS:
https://nodejs.org/en/download/package-manager


- install packages
run `npm install`

- 進入前端專案資料夾
run `cd frontend`

- start the project
run `npm start`

- check 
check on "http:localhost:3000"
